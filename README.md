# 项目开发文档
mary-vue
## 项目运行
```
yarn install
yarn run serve
yarn run build
```
## 项目书写规范
```
1.多组件数据共享才能用VUEX;
2.watch监听函数尽量避免使用;
3.配置信息统一放置在config文件中;
4.公共组件放在src/components文件中，注意公共组件中slot插槽，依据组件的方法不建议写具体操作，具体操作应在父组件中写；
```
